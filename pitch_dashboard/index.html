Innovation

How It Works

Three key innovations that make this solution production-ready.

🧠

Design-Time AI Clustering

We call Claude once at design time to cluster 133 raw endpoints into 19 semantic workflows. The AI only pays the cognitive cost once — not on every user request.

⚡

Runtime Execution Engine

A pure-Python workflow runner reads the YAML blueprint and executes step-by-step HTTP calls, handles conditions, loops, variable chaining — zero LLM calls at runtime.

🎛️

Hierarchical Exposure

When a high-level workflow hits an edge case, Claude calls list_raw_endpoints to reveal the underlying Redfish calls. It drops to surgical precision only when needed.

Integration

Connect to Claude Desktop

Add this snippet to your claude_desktop_config.json and restart Claude.

claude_desktop_config.json
{
  "mcpServers": {
    "redfish-workflow-proxy": {
      "command": "python",
      "args": ["C:\\path\\to\\MCP-Workflow\\mcp_server.py"],
      "env": {}
    }
  }
}

Add Config & Restart
